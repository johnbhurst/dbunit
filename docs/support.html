Known Issues

  • Dbunit only support one database schema per connection. Multiple schemas support is currently in progress.
  • Dbunit do not support binary data types yet. Since version 1.2 columns with unsupported data types are ignored and are not causing a DataTypeException to be thrown anymore. Support for BINARY, VARBINARY and LONGVARBINARY columns is planned.
  • Browse Dbunit bug database.
